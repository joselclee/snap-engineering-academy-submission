HOME ABOUT CARS
This website was made purely with vanilla HTML, CSS, and JavaScript from the starter code provided by Snapchat Engineering Academy.

Objective: The website is developed for the 2024 Cohort by Jose Leandre Castillo Lee.

Description: The website shows an interactive display of Jose's favorite cars. The cards have detailed information about each car, which is shown when hovering over the card. The cards can also be removed by clicking the "-" button on the top right corner of the card. Once the card is removed, the site adds a new card from the catalog of cars until there are no more cars to display.

Pages on the site:
  • Home: The main page of the site, which displays the cards of the cars.
  • About: This page, which provides information about the site and its developer.
  • Cars: The page where the user can add a new car to the catalog and see all the cars in the databank.
